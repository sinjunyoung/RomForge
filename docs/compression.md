# 압축 / 압축 해제

다양한 압축 포맷을 지원하는 탭입니다.

- 원본 파일을 넣으면 자동으로 압축됩니다.
- 압축된 파일을 넣으면 자동으로 압축이 해제됩니다.
- 지원 포맷은 화면의 안내를 참고하세요.

## 화면 구성

1. **Hint & 드래그 앤 드롭 존**
2. **파일 추가** — 여러 개 동시 지정 가능
3. **폴더 추가** — 폴더 하위 파일을 재귀적으로 전체 추가 (`D:\`처럼 드라이브 전체를 지정하면 그 안의 전체 파일이 추가됨)
4. **선택 파일 삭제**
5. **추가된 전체 파일 삭제**
6. **압축 설정 탭으로 이동**
7. **압축 / 해제 시작**
8. **취소** — 취소하면 진행 중이던 파일은 삭제됨
9. **로그**

## 압축 설정

### CHDMAN

**압축 알고리즘**

PS2용 안드로이드 에뮬레이터(에테르 등)와의 호환을 위해 zlib이 필요한 경우가 아니라면 **zstd**를 사용하면 됩니다. 그 외 세부 옵션은 화면에 나온 그대로이므로 별도 설명은 생략합니다.

<img width="849" height="176" alt="1783853964" src="https://github.com/user-attachments/assets/fe6ea570-0abd-4509-8f42-8f7cb8aa32eb" />

### Switch

**압축 레벨**

zstd 계열은 압축 레벨을 0~22까지 지정할 수 있지만, RomForge에서는 3~18까지만 지정 가능합니다. 19 이상은 속도가 급격히 느려지는 데 비해 압축률 차이는 미미해서 제외했습니다.

**압축 모드**

- **블록**: 탐색 가능, 고속 — 에뮬레이터에서 사용 가능
- **블록리스**: 탐색 불가, 고압축 (블록 대비 압축률 차이는 크지 않음)

**압축 무결성 검증**

압축 후 원본과 데이터가 동일한지 BLAKE3 해시로 검증합니다. 다른 플랫폼은 크게 문제가 없지만, Switch는 현세대 기기라 포맷이 바뀔 가능성이 있으므로 검증을 권장합니다. 검증 없이 원본을 삭제했다가 복구가 불가능해질 수 있습니다.
