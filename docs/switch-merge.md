# Switch - 병합 / 분리

본편 + 업데이트 + DLC를 하나로 병합하거나, 병합된 파일을 다시 본편 + 업데이트 + DLC로 분리하는 탭입니다.

## 화면 구성

1. **파일 리스트** — 병합 시에는 관련 파일들을 한꺼번에 넣으면 자동으로 그룹핑해서 병합합니다.
2. **파일 추가** — 여러 개 동시 지정 가능
3. **폴더 추가** — 폴더 내 전체 파일 추가
4. **선택 삭제**
5. **전체 삭제**
6. **압축 설정 탭으로 이동**
7. **출력 구분** — 결과물을 NSP로 만들지, XCI로 만들지 선택
8. **출력 폴더** — 최종 파일이 생성될 폴더
9. **언팩** — 롬 파일을 분해
10. **리팩** — 분해한 폴더를 NSP로 다시 패킹
11. **로그**

## 사용 예시

- **본편 + 업데이트 + DLC를 합칠 때**
- **통합 파일에서 본편 + 업데이트 + DLC를 분리할 때**
- **본편 + 업데이트(1.1) 통합 파일에 업데이트(1.2)를 합칠 때**: 기존 업데이트(1.1)는 제거되고 본편 + 업데이트(1.2)로 병합됩니다. 즉, 최신 업데이트만 남습니다.
