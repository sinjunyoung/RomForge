# Switch - 포맷 변환

Switch용 포맷 변환 기능입니다. NSP, NSZ, XCI, XCZ를 상호 변환할 수 있습니다.

파일마다 개별적으로 출력 포맷을 지정해서 변환할 수 있습니다.

## 화면 구성

1. **파일 리스트**
2. **파일 별 출력 포맷 지정**
3. **파일 추가** — 여러 개 동시 지정 가능
4. **폴더 추가** — 폴더 내 전체 파일 추가
5. **선택 삭제**
6. **전체 삭제**
7. **시작**
8. **취소**
9. **로그**

## NSZ, XCZ 변환 시 영향받는 설정

**압축 레벨**

zstd 계열은 압축 레벨을 0~22까지 지정할 수 있지만, RomForge에서는 3~18까지만 지정 가능합니다. 19 이상은 속도가 급격히 느려지는 데 비해 압축률 차이는 미미해서 제외했습니다.

**압축 모드**

- **블록**: 탐색 가능, 고속 — 에뮬레이터에서 사용 가능
- **블록리스**: 탐색 불가, 고압축 (블록 대비 압축률 차이는 크지 않음)

**압축 무결성 검증**

압축 후 원본과 데이터가 동일한지 BLAKE3 해시로 검증합니다. Switch는 현세대 기기라 포맷이 바뀔 가능성이 있으므로 검증을 권장합니다. 검증 없이 원본을 삭제했다가 복구가 불가능해질 수 있습니다.

## 사용 예시

NSZ를 받았는데 XCZ로 바꾸고 싶을 때 사용합니다. (다른 확장자는 안 되고 모든 파일을 XCZ로 통일하고 싶은 경우에 유용합니다.)
