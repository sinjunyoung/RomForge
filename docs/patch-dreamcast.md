# 패치 - 드림캐스트

드림캐스트 패치 전용(DCP) 탭입니다.

## 화면 구성

1. **패치 대상 원본** — GDI 파일
2. **패치 파일** — DCP 파일
3. **패치 후 자동 압축 기능** — 체크 시 CHD로 자동 압축
4. **패치 진행 바**
5. **초기화** — 1, 2번 항목을 초기화합니다 (잘못 선택했거나 다른 게임을 이어서 패치할 때 사용)
6. **해시 계산** — 1번에 선택된 원본 파일의 해시 값을 계산합니다 (GDI 파일 특성상 큰 의미는 없습니다)
7. **패치 시작**
8. **패치 취소** — 진행 중인 패치를 취소하고, 만들어지던 파일은 삭제합니다
9. **로그**
